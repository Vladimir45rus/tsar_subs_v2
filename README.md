# 💳 Царь Подписки

Сервис для централизованного управления цифровыми подписками. Помогает отслеживать списания, прогнозировать расходы и вовремя отменять ненужные подписки. Часть экосистемы «Царь».

## 🎯 Функциональность

- Добавление подписок по названию сервиса (YouTube, Netflix, Яндекс.Плюс и др.)
- Автоматическое прогнозирование дат списаний
- Уведомления о скором списании (email / push)
- Статистика расходов по категориям
- Историй операций и возможность отмены подписки

## 🛠 Технологии

- **Фронтенд:** React (Vite)
- **Бэкенд:** Python + FastAPI (планируется)
- **Хранилище:** локальное (IndexedDB) / в перспективе — облачное
- **Сборка:** Vite, npm

## 📦 Статус разработки

✅ Прототип фронтенда (добавление подписок, дат, сумм)  
✅ Сохранение данных в браузере  
🔄 Интеграция с Царь ID для авторизации  
⚙️ Бэкенд-часть (синхронизация между устройствами)
